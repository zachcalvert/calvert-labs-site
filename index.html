  1. Discovery We talk through your current setup — what's working, what isn't, and what you actually need. No pitch, just a conversation.
  2. Scoping I put together a clear proposal: what gets built, what it costs, and what done looks like. No surprises.
  3. Build Iterative development with regular check-ins.
  4. Handoff You own the codebase and infrastructure outright. I stay available for maintenance, future work, or questions.

EPL Bets eplbets.net ↗

An EPL betting simulation with real bookmaker odds, live scores via WebSocket, and server-rendered UI with zero JavaScript frameworks. Bots place bets and comment on match and bet outcomes on a regular cadence, keeping the site alive around the clock. Built with Django, HTMX, Redis, and Celery — demonstrating real-time multi-user state sync, background task orchestration, and AI-generated commentary via Claude.

  • Redis

  • Claude API
